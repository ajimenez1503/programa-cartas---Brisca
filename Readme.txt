#Programa de cartas : Brisca

(C) Programmed by: Antonio Jimenez Martínez
 


*Version:0.2*

El programa esta compuesto por las siguientes clases relacionadas entre si:

*   Una clase cartas.
*   Una clase baraja.
*   Una clase partida
*   Una clase jugador.

Este programa administra una partida de cartas (BRISCA) 
entre dos jugadores, los usuarios eligen que jugada es 
la mas idonea y el programa administra las carta y la 
puntuación.

Existe diferentes tipos de juego:

*   un usuario contra un usuario.
*   una maquina aleatoria contra un usuario.
